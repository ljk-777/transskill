## Phase 4 完成：MCP JSON + SOUL.md Parser

### 新增
- **MCPJsonParser**: 解析 `{ name, description, tools[] }` → `IntermediateSkill`，tools 完整保留在 `platformSpecific.mcp.tools`，自动生成可读的 instructions
- **SOULMdParser**: 解析 SOUL.md（OpenClaw 身份文件），提取标题、章节、规则
- **MCPJsonRenderer**: `IntermediateSkill` → MCP JSON 格式输出，含 tools 转换
- FormatDetector 增强：通过文件名 (`SOUL.md`) 和内容 `# SOUL.md` 签名自动检测

### 更新的格式支持
```
MCP JSON → SKILL.md, .cursorrules, .mdc
SOUL.md  → SKILL.md, .cursorrules, .mdc
```
